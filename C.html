基础知识讲解

二叉树

每个节点最多两个子节点。C 语言用 struct Node 与指针 left、right 表示。

栈（Stack）

后进先出（LIFO）的线性结构。非递归遍历时借助栈保存待处理节点，核心操作：push、pop。

先序遍历

访问顺序：根 → 左 → 右。将根压栈，循环弹出访问，再依次将右子、左子压栈。

中序遍历

访问顺序：左 → 根 → 右。沿左路将节点压栈，栈空弹出访问，再转向右子树。

后序遍历

访问顺序：左 → 右 → 根。双栈法：S1 压出的节点全部按顺序压入 S2，S2 弹出顺序即后序。

时间复杂度

三种遍历均为 O(n)，每个节点恰好访问一次。空间复杂度平均 O(log n)，退化链最坏 O(n)。
